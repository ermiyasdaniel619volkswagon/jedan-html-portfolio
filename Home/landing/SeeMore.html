  1. <header></header>
    When to use: Use this for introductory content or a set of navigational links.
    See More Where to use: Usually at the very top of a page

  2. <div></div>
    When to use: Use this for introductory content or a set of navigational links.
    Where to use: Usually at the very top of a page

  3. <html></html>
    When to use: To wrap the entire document content.
    Where to use: The root of every HTML page.

  4. <head></head>
    When to use: To contain machine-readable information (metadata).
    Where to use: Right after the opening html tag.

  5. <title></title>
    When to use: To define the name of the page shown in the browser tab.
    Where to use: Inside the head section.

  6. <body></body>
    When to use: To contain all the visible content of a webpage.
    Where to use: After the closing head tag.

  7. <nav></nav>
    When to use: For a block of major navigation links.
    Where to use: Usually inside the header or sidebar.

  8. <main></main>
    When to use: For the unique primary content of the page.
    Where to use: Between the header and the footer.

  9. <section></section>
    When to use: For grouping thematic content together.
    Where to use: Inside the main tag.

  10. <article></article>
    When to use: For self-contained content like blog posts.
    Where to use: Inside a section or the main tag.

  11. <aside></aside>
    When to use: For content indirectly related to the main text (sidebars).
    Where to use: Next to the main content.

  12. <footer></footer>
    When to use: For copyright info, contact details, or site maps.
    Where to use: At the bottom of a page or section.

  13. <h1> to <h6>
    When to use: To create headings of different importance.
    Where to use: At the start of new sections.

  14. <p></p>
    When to use: To define a block of text as a paragraph.
    Where to use: Inside articles, sections, or divs.

  15. <ul></ul>
    When to use: To create an unordered (bulleted) list.
    Where to use: Wherever you need to list items without specific order.

  16. <ol></ol>
    When to use: To create an ordered (numbered) list.
    Where to use: For steps or ranked items.

  17. <li></li>
    When to use: To define an individual item within a list.
    Where to use: Inside ul or ol tags.

  18. <a></a>
    When to use: To create hyperlinks to other pages or sections.
    Where to use: Inside paragraphs, navs, or lists.

  19. <img />
    When to use: To embed an image in the document.
    Where to use: Inside any content-holding tag.

  20. <strong></strong>
    When to use: To give text strong importance (usually bold).
    Where to use: Inside sentences or paragraphs.

  21. <em></em>
    When to use: To emphasize text (usually italics).
    Where to use: Inside sentences.

  22. <blockquote></blockquote>
    When to use: For long quotes from another source.
    Where to use: Inside the main content area.

  23. <code></code>
    When to use: To display a fragment of computer code.
    Where to use: Inside paragraphs or pre tags.

  24. <pre></pre>
    When to use: To show preformatted text (preserves spaces).
    Where to use: For displaying code blocks or ASCII art.

  25. <span></span>
    When to use: To style a small part of inline text.
    Where to use: Inside any text-level element.

  26. <br />
    When to use: To force a line break in text.
    Where to use: Inside paragraphs or addresses.

  27. <hr />
    When to use: To create a thematic break or horizontal line.
    Where to use: Between sections or articles.

  28. <button></button>
    When to use: To create a clickable element to perform an action.
    Where to use: Inside forms or UI components.

  29. <table></table>
    When to use: To display data in a grid format.
    Where to use: Inside the main content area.

  30. <form></form>
    When to use: To collect user input.
    Where to use: Usually in contact sections or login areas.

  31. <input />
    When to use: To create various types of input fields (text, checkbox).
    Where to use: Inside a form tag.

  32. <label></label>
    When to use: To provide a caption for an input field.
    Where to use: Next to or wrapping an input tag.
